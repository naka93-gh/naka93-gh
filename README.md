# こんにちは 👋

Web系エンジニアです。使いたいものを自分で作っています。

## About

- デスクトップアプリの開発が好きで、Electron / Tauri / Electrobun など色々なフレームワークを試しています。
- 開発者向けのツールや、業務を効率化するツールを作っています。
- TypeScript をメインに、Rust も書きます。
